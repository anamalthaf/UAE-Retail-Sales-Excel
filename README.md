# UAE Retail Sales Analysis (Excel)

## Project Overview
This project analyzes retail sales data using Microsoft Excel to uncover sales trends, regional performance, and top-performing products. The goal is to demonstrate practical data analysis skills including data cleaning, pivot tables, and dashboard creation.

## Tools Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Excel Dashboards

## Dataset
- Source: Kaggle (Superstore-style retail sales data)
- Format: CSV (converted to Excel)
- Records include order dates, regions, product categories, and sales values.

## Data Preparation
- Converted raw CSV data to Excel format
- Removed duplicate records
- Verified and standardized date and numeric fields
- Cleaned and standardized text fields (region, category, product names)

## Analysis Performed
- Monthly sales trend analysis
- Sales performance by region
- Sales performance by product category
- Identification of top 10 products by total sales

## Dashboard
The Excel dashboard provides:
- Monthly sales trend visualization
- Regional sales comparison
- Category-level sales breakdown
- Top 10 products by sales

## Key Insights
- Sales show clear variation across regions and product categories
- A small number of products contribute a significant portion of total sales
- High-value office equipment dominates top product sales
